# IdeaMarket - Setup & Deployment Guide

## Quick Start (5 minutes)

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Open browser to `http://localhost:5173`

### 3. Try the App
- Click "Try Demo Account" to login instantly
- Browse ideas on the Explore page
- Create your own idea from the dashboard
- Purchase ideas and leave reviews

## Development Guide

### Project Structure
- `src/App.tsx` - Main app with routing
- `src/pages/` - Page components
- `src/components/` - Reusable components
- `src/store/` - Zustand state management
- `public/` - Static assets

### State Management (Zustand)

#### Auth Store
```typescript
import { useAuthStore } from './store/authStore';

// Get user
const user = useAuthStore((state) => state.user);

// Login
useAuthStore((state) => state.login)(user, token);

// Logout
useAuthStore((state) => state.logout)();
```

#### Idea Store
```typescript
import { useIdeaStore } from './store/ideaStore';

// Get all ideas
const ideas = useIdeaStore((state) => state.ideas);

// Add idea
useIdeaStore((state) => state.addIdea)(newIdea);

// Get idea by ID
const idea = useIdeaStore((state) => state.getIdeaById)('idea-id');

// Purchase idea
useIdeaStore((state) => state.purchaseIdea)('idea-id', 'user-id');

// Add rating
useIdeaStore((state) => state.addRating)('idea-id', rating);
```

### Creating New Pages

1. Create file in `src/pages/`
2. Add route in `src/App.tsx`
3. Import page component

Example:
```typescript
// src/pages/MyPage.tsx
export default function MyPage() {
  return <div>My Page</div>;
}

// In App.tsx
import MyPage from './pages/MyPage';

// In Routes
<Route path="/my-page" element={<MyPage />} />
```

### Creating New Components

1. Create file in `src/components/`
2. Use TypeScript for props
3. Style with Tailwind classes

Example:
```typescript
// src/components/MyComponent.tsx
interface MyComponentProps {
  title: string;
  onClick: () => void;
}

export default function MyComponent({ title, onClick }: MyComponentProps) {
  return (
    <button className="px-4 py-2 bg-orange-500 text-white rounded-lg">
      {title}
    </button>
  );
}
```

## Building for Production

### 1. Build
```bash
npm run build
```
Output: `dist/index.html`

### 2. Preview Build
```bash
npm run preview
```

### 3. Deploy
The `dist` folder contains a single HTML file ready for deployment.

**Option A: Vercel (Recommended)**
```bash
npm install -g vercel
vercel
```

**Option B: Netlify**
1. Connect GitHub repo to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `dist`

**Option C: Traditional Hosting**
- Upload `dist/index.html` to your web server
- Configure server to serve `index.html` for all routes

## Backend Integration (Future)

### 1. Update API Calls
Replace mock functions with real API calls using `axios`:

```typescript
import axios from 'axios';

const API_URL = import.meta.env.VITE_API_URL;

// Login
const response = await axios.post(`${API_URL}/auth/login`, { email, password });
const { token, user } = response.data;
useAuthStore((state) => state.login)(user, token);

// Get ideas
const response = await axios.get(`${API_URL}/ideas`);
useIdeaStore((state) => state.setIdeas)(response.data);

// Create idea
const response = await axios.post(`${API_URL}/ideas`, ideaData, {
  headers: { Authorization: `Bearer ${token}` }
});
```

### 2. Set Authorization Headers
```typescript
const token = useAuthStore((state) => state.token);
const config = {
  headers: { Authorization: `Bearer ${token}` }
};
axios.get('/api/protected', config);
```

### 3. Handle Errors
```typescript
try {
  const response = await axios.get('/api/ideas');
  // handle success
} catch (error) {
  if (error.response?.status === 401) {
    // Redirect to login
    useAuthStore((state) => state.logout)();
  }
}
```

## Troubleshooting

### Port Already in Use
```bash
# Kill process on port 5173
# Windows
netstat -ano | findstr :5173
taskkill /PID <PID> /F

# macOS/Linux
lsof -ti:5173 | xargs kill -9
```

### Node Modules Issues
```bash
rm -rf node_modules package-lock.json
npm install
```

### Build Errors
```bash
npm run build
```
Check the error message and ensure all imports are correct.

### State Not Updating
- Check Zustand store subscriptions
- Verify state update functions are called
- Use React DevTools to inspect state
